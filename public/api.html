Overview

The SpinGuard CAPTCHA API allows developers to generate and verify CAPTCHA codes. It helps protect forms from automated submissions. This API is lightweight, easy to use, and perfect for web-based applications.

API Endpoints

1. Generate CAPTCHA

Endpoint: GET /api/captcha

Description: Generates a new CAPTCHA with a unique ID and code.

Response Example:

{
    "captchaId": "e6a5f5a9-8b0e-46e2-b089-14f6038a2dbf",
    "captcha": "3GK7P9L"
}

2. Verify CAPTCHA

Endpoint: POST /api/verify

Description: Verifies the user’s input against the generated CAPTCHA code.

Request Body Example:

{
    "captchaId": "e6a5f5a9-8b0e-46e2-b089-14f6038a2dbf",
    "userInput": "3GK7P9L"
}

Response Example (Success):

{
    "success": true,
    "message": "CAPTCHA verified successfully."
}

Frontend Integration Example

Here’s how you can integrate the SpinGuard CAPTCHA API in a form:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>CAPTCHA Integration Example</title>
</head>
<body>
    <form id="captchaForm">
        <label>Email:</label>
        <input type="email" id="email" required>
        <br><br>

        <label>CAPTCHA: <span id="captchaDisplay"></span></label>
        <button type="button" onclick="getCaptcha()">Generate CAPTCHA</button>
        <br><br>

        <input type="text" id="captchaInput" placeholder="Enter CAPTCHA" required>
        <br><br>
        <button type="submit">Submit</button>
    </form>

    <script>
        let captchaId = '';

        async function getCaptcha() {
            const response = await fetch('https://your-app-name.herokuapp.com/api/captcha');
            const data = await response.json();
            captchaId = data.captchaId;
            document.getElementById('captchaDisplay').innerText = data.captcha;
        }

        document.getElementById('captchaForm').addEventListener('submit', async function (e) {
            e.preventDefault();
            const userInput = document.getElementById('captchaInput').value;

            const response = await fetch('https://your-app-name.herokuapp.com/api/verify', {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({ captchaId, userInput })
            });

            const data = await response.json();
            if (data.success) {
                alert('CAPTCHA verified! Form submitted.');
            } else {
                alert('Invalid CAPTCHA. Try again.');
            }
        });
    </script>
</body>
</html>

Testing the API

You can use tools like Postman or cURL to test the API endpoints:

Generate CAPTCHA (cURL):

curl https://spinguard-api.herokuapp.com/api/captcha

Verify CAPTCHA (cURL):

curl -X POST https://spinguard-api.herokuapp.com/api/verify \
-H "Content-Type: application/json" \
-d '{"captchaId": "unique-captcha-id", "userInput": "3GK7P9L"}'

Error Handling

  • 404 Not Found: CAPTCHA ID not found or expired.
  • 400 Bad Request: Malformed request body.
  • 200 OK: CAPTCHA verified successfully.
